Welcome to Bootstrap

Bootstrap is one of the most popular frameworks for building responsive and mobile-first websites. It provides ready–made CSS classes, components, and layouts that make web development faster and easier. In this simple tutorial page, we look at how Bootstrap works and how you can start using it to create clean and professional web pages.

What You Will Learn

In these Bootstrap tutorials, you will learn how the grid system works, how to use rows and columns, and how to build layouts that adjust automatically on different screen sizes. Bootstrap also includes many reusable UI components like buttons, navbars, dropdowns, cards, and forms.

Everything is designed to help beginners and experienced developers build strong and attractive interfaces without writing everything from scratch.

Why Bootstrap?

Bootstrap is easy to learn, widely supported, and works well for both small and large projects. It also saves time because you don’t need to create every design element manually.

Bootstrap Buttons

Bootstrap comes with several button styles you can use without writing any CSS. The main classes include btn-primary, btn-secondary, btn-success, btn-danger, and more.

Example:

To create a button, simply use:
<button class="btn btn-primary">Click</button>

Navbar Tutorial

The navbar is one of the most important navigation components in Bootstrap. It supports branding, links, toggles for mobile view, and dropdowns.

A typical navbar uses:

<nav class="navbar navbar-expand-lg navbar-dark bg-dark">

You can customize it using classes like navbar-light, bg-primary, navbar-expand-lg and more.

Bootstrap Grid System

The grid system is the backbone of all Bootstrap layouts. It divides the page into 12 columns. You can combine these to create any layout.

Bootstrap Containers

Containers help center and limit the width of your content. Bootstrap has three main container types:

  • .container – fixed width
  • .container-fluid – full width
  • .container-lg – responsive width

How Columns Work

Columns must always be placed inside a row. A full row has 12 units, so you can divide it any way you want.

Example:

<div class="row">
  <div class="col-8">Main</div>
  <div class="col-4">Side</div>
</div>

Spacing Classes

Bootstrap spacing classes make it easy to add margin or padding. Use m for margin and p for padding.

Examples:

  • mt-3 → margin-top
  • mb-4 → margin-bottom
  • p-2 → padding
  • px-4 → left & right padding

Values go from 0 to 5.

Notes

Remember: Bootstrap uses a 12-column grid layout. When building a page, think in terms of rows and columns. This structure helps keep everything clean and aligned.

Tips

Always include container, row, and col- classes when working with Bootstrap layout. It keeps the design responsive.
